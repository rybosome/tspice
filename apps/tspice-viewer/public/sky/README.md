# Sky textures

This directory contains background sky textures used by the viewer.

## `starmap_4k.jpg` (active)

- **Resolution:** 4096×2048
- **File size:** ~2.3 MB
- **Source:** NASA Scientific Visualization Studio (SVS)
  - Page: https://svs.gsfc.nasa.gov/3895 ("Deep Star Maps")
  - Direct asset: https://svs.gsfc.nasa.gov/vis/a000000/a003800/a003895/starmap_4k.jpg
- **Credit / license:**
  - NASA SVS content is generally **public domain** unless otherwise noted.
  - NASA media usage guidelines apply: https://www.nasa.gov/nasa-brand-center/images-and-media/

## `starmap_2020_1k.jpg` (legacy, kept for reference)

- **Resolution:** 1024×512
- **Source:** NASA Scientific Visualization Studio (SVS)
  - Page: https://svs.gsfc.nasa.gov/4851/ ("Deep Star Maps 2020")
  - Direct asset used: https://svs.gsfc.nasa.gov/vis/a000000/a004800/a004851/starmap_2020_4k_print.jpg
- **Notes:**
  - The upstream SVS download labeled `starmap_2020_4k_print.jpg` is actually only 1024×512.
