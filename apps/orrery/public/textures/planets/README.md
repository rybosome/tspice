# Planet texture credits

This directory contains low-resolution textures used by the Orrery demo.

Most textures here are **256×128** PNGs (lightweight demo goal), but a few Earth-specific layers are higher-resolution to support atmosphere / clouds / night lights.

- `mars-mola-normal-2k.png` / `mars-roughness-proxy-2k.png`
  - Source DEM: **Mars Global Surveyor (MGS) MOLA MEGDR** (Mission Experiment Gridded Data Record), 16 pixels/degree (0.0625°).
    - PDS Geosciences Node (dataset: `MGS-M-MOLA-5-MEGDR-L3-V1.0`)
    - DEM IMG (binary): https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg016/megt90n000eb.img
    - Label (metadata): https://pds-geosciences.wustl.edu/mgs/mgs-m-mola-5-megdr-l3-v1/mgsl_300x/meg016/megt90n000eb.lbl
  - Generated by: `apps/orrery/scripts/generate-mars-maps.js`
  - Notes:
    - **Normal map** is derived from DEM central differences. The longitudinal derivative is scaled by `cos(latitude)`.
    - **Roughness proxy** is a heuristic combining DEM slope + albedo luminance (from `mars-viking-colorized-4k.jpg`) so Mars stays mostly matte with subtle variation:
      - `roughness = clamp(base + amp * (wSlope * slopeNorm + wAlbedo * (1 - luma)), 0, 1)`
